# ELO算法以及可视化实现


1. 如何计算等级分？
    ```
    ELO算法
    ```
2. 如何保证每人参赛次数尽量相等？
    ```
    1. 设置一个标志，记录参赛次数，每次取最小的参赛次数记录L
    ```
3. 如何匹配对手？
    ```
    根据第二条原则产生一个随机数i
    1. i>0.5
        选择第一个大于等于L等级分的记录
    2. i<0.5
        选择第一个小于等于L等级分的记录
    ```
4. 如何进行展示？
    ```
    在窗口中进行匹配参比
    每一轮重新两位选手的等级分并存进数据库
    触动查看排行生成当前排行版
    ```
  
 5. 其他
    1. 数据库连接以及文件夹路径在src\setting.json设置
    2. 运行rebuild_db.py可对数据库存储的数据进行重建
